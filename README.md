## Requirements

* ruby >= 1.9.3
* Some database (sqlite3/mysql/postgresql should all work)
* [A day-to-day Jeopardy calendar](http://www.amazon.com/Jeopardy-2014-Day---Day-Calendar/dp/1449430880/)
